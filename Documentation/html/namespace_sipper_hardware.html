Detailed Description

Contains Classes that are specific to SIPPER's physical characteristics such as Instrument Data.

Classes that represent Instruments, SIPPER File Formats, Cruises, Stations, and Deployments are grouped here. The PICES software installation specifics are also dealt within this namespace.
