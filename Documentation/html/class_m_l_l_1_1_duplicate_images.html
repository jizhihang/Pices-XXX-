Detailed Description

Detects duplicate images in a given FeaureVectorList objects.

Author
Kurt Kramer

Will derive a list of duplicate FeatureVector objects in a given list. It will use both the Image File Name and feature data to detect duplicates. A duplicate can be detected in two ways. If two or more entries have the same ImageFileName or FeatureData.

The simplest way to use this object is to create an instance with a FeatureVectorList object that you are concerend with. Then call the method DupImages (), which will return the list of dupliactes found via a structure called DuplicateImageList.
