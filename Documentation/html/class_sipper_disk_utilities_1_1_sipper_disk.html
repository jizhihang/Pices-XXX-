Detailed Description

SipperDisk is used for reading and writting to a SIPPER disk drive.

The SIPPER disk has a very simple format who's details are described below. You read the Master Boot record and get the 1st sector from "Partition Entry 1". The SIPPER Partition will start with one Partition decriptive block followed by 2048 File Control Block, on efor each SIPPER file created. The File control blocks will be followed by the actual SIPPER data. SIPPER records data sequentially starting at the begiining of the drive until the end of the drive. There will be not fragmentation issues to deal with. Each SIPPER file will have a start and end sector that is specified in its corresponding File Control Block.
