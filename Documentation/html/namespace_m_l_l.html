Detailed Description

Namespace for all Machine Learning and SIPPER specific code.

Author
Kurt Krame * Date: 2009-04-16 *

A library of routines used for Machine Learning, building Classifiers, etc. It also includes code that is specific to PICES and SIPPER. At som point ion the future we need split this library into three libraries. MachineLearning, DataBase, and PicesRoutines.

This library covers three different areas that at some time in the future should be made into three seperate libraries.
1) Generic Machine Learning code.
2) Database Related Code.
3) Analysis.
4) PICES specific code that would be built onto of 1 and 2.

"Machine Learning Code"

"Feature Data"

There are several Feature Data formats supported. Each one has its own class that is derived from 'FeatureFileIO'. The decription of the data is managed by 'FileDesc'. For each type of dataset there will exist on one instance of a FileDesc class.
