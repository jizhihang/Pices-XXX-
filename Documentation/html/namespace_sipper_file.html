Detailed Description

Routines for managing SIPPER specific classes, c# equivelent of the c++ SipperInstruments namepace.

This name space contaions c# equielents to the classes c++ classes in SipperInstruments. These classes specialize on SIPPER specific items such as the Sipper File Format or Sipper Instrument Data.
