Detailed Description

A class that will consist of static metods only.

This is where we will keep track of the current active training libraries. Thios way we will not need to implement the code in each seperate application.

Once a model is loaded via "SetModel1" or "SetModel2" they will not be unloaded except by the Memory Manager. This way if a process is using a specific model that was retrieved by "Model1" or "Model2" they can continue to use it even if a new model is set by "SetModel1" or "SetModel2".
