Loads an SVM model from disk.

The '_cancelFlag' parameter is meant to allow another thread to cancel processing by a different thread in SVMmodel. Ex: If one thread is building a new SVM for this instance of SVMmodel, that thread will periodcaly monitor '_cancelFlag', if it is set to true it will terminate its processing.

Parameters
[in]_inA file stream to read and build SVMModel from.
[out]_successfulSet to true if the model is succesfulyl loaded, false otherwise
[in]_fileDescA description of the data file. I'm not sure this is needed for this function.
[in]_logA logfile stream. All important events will be ouput to this stream
[in]_cancelFlagIf set to true any process running in SVMModel will terminate.
