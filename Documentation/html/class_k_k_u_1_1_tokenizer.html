Detailed Description

Class is meant to break down a stream into a set of logical tokens.

Author
Kurt Kramer

This class was originally created while taking Non Linear Systems. It breaks up a source KKStr or text file into logical tokens. You can create your own source of characters by creating a Class derived from MLL::TokenBuffer.
