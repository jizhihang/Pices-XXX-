Detailed Description

This is a vesion 2.89 of "Chih-Chung Chang" and "Chih-Jen Lin" libSVM (Support Vector Machine) that selects different features per pair of classes.

Downloaded from "http://www.csie.ntu.edu.tw/~cjlin/libsvm/"

For each pair of classes a 'FeatureNumList" instance specifies which features are to be used during kernal computations.
