System::String^  imageFileName,
PicesRunLog^  log

Calculates features off the image that is in "raster".

Features that are based off instrumentation data will come from the IntrumentDataManager the "imageFileName" will be parsed to get SipperFileName and ScanLine.

Parameters
[in]rasterRaster in the form of a 2 dmensional byte array.
[in]imageFileNameName of image that is stored in 'raster'; should follow Sipper file name conventions.
[in,out]loggingfile.
