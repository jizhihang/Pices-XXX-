Detailed Description

Responsable for keeping track of a list of jobs.

See the application called RandomSplitsJobManager. It is the first alplication to use this library. any class that is derived from this class are required to implement these methid "StatusFileProcessLine", "StatusFileProcessLineJobStatusChange", "ToStatusStr"
